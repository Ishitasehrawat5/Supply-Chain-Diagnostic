# Supply-Chain-Diagnostic
Supply Chain Performance Diagnostic

A consulting-style supply chain analysis for a Fashion & Beauty startup using Excel.

Objective: Identify supply chain inefficiencies and recommend cost and quality improvements.

Tools Used: Microsoft Excel, Pivot Tables, Data Analysis

Key Findings:

Skincare drives 42% of revenue but has the highest defect rates
Supplier 3 is both the slowest and lowest quality supplier
Route C shipping costs are 10% higher than Route A

Recommendations:

Implement quality control checks at Supplier 3 and Supplier 5
Renegotiate lead times with Kolkata-based suppliers
Shift shipping volume from Route C to Route A

Expected Impact: 15-20% margin improvement and 25% reduction in lead times
